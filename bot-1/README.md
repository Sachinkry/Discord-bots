## LW-3 Faucet Bot: A discord-bot to send test token funds 
### Project-info
On selecting/typing one of the commands:-
`/faucet goerli eth`
`/faucet goerli link`
`/faucet mumbai matic`
`/faucet mumbai link`   
`/faucet alfajores celo`
the bot sends funds.
